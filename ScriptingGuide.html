Scripting API Documentation

Move

Moves the entity in 3D space.

Parameters: float x, float y, float z

Returns: void

entity:Move(1.0, 0.0, 0.0)

SetPos

Sets the absolute position of the entity.

Parameters: float x, float y, float z

Returns: void

entity:SetPos(0.0, 0.0, 0.0)

Rotate

Rotates the entity around its local axes.

Parameters: float x, float y, float z

Returns: void

entity:Rotate(0.0, 90.0, 0.0)

Scale

Scales the entity in 3D space.

Parameters: float x, float y, float z

Returns: void

entity:Scale(2.0, 2.0, 2.0)

ApplyForce

Applies a world-space force to the entity.

Parameters: float x, float y, float z

Returns: void

entity:ApplyForce(0.0, 10.0, 0.0)

ApplyRelativeForce

Applies a local-space force to the entity.

Parameters: float x, float y, float z

Returns: void

entity:ApplyRelativeForce(0.0, 0.0, 5.0)

ChangeColor

Changes the color of the entity.

Parameters: float r, float g, float b

Returns: void

entity:ChangeColor(1.0, 0.0, 0.0)

RotateCamera

Rotates the camera around its axis.

Parameters: float pitch, float yaw, float roll

Returns: void

entity:RotateCamera(0.0, 45.0, 0.0)

SetCameraRot

Sets the absolute camera rotation.

Parameters: float pitch, float yaw, float roll

Returns: void

entity:SetCameraRot(0.0, 90.0, 0.0)

MoveCamera

Moves the camera by an offset.

Parameters: float x, float y, float z

Returns: void

entity:MoveCamera(0.0, 5.0, -10.0)

SetCameraPos

Sets the camera's world position.

Parameters: float x, float y, float z

Returns: void

entity:SetCameraPos(10.0, 20.0, 30.0)

ChangeAnimation

Changes the current animation of the entity.

Parameters: const char* animName

Returns: void

entity:ChangeAnimation("Run")

AddScript

Adds a script to the entity.

Parameters: const char* scriptName

Returns: void

entity:AddScript("enemy_ai")

GetEntity

Returns a reference to another entity by name.

Parameters: const char* name

Returns: Entity Object

local player = GetEntity("Player")

SpawnObject

Spawns a new object in the scene.

Parameters: const char* name, uint8_t type

Returns: Entity Object

local obj = SpawnObject("Crate", 1)

LoadObject

Loads a new object from file or prefab.

Parameters: const char* path, const char* type

Returns: Entity Object

local tree = LoadObject("tree.obj",
    "model")

IsKeyDown

Checks if a key is currently pressed.

Parameters: string key

Returns: bool

if IsKeyDown(87) then
  entity:Move(0, 0, 1)
end

Pos.getX / getY / getZ

Gets entity position along specified axis.

Parameters: none

Returns: float

local x = Pos:getX()

Rot.getX / getY / getZ

Gets entity rotation along specified axis.

Parameters: none

Returns: float

local yRot = Rot:getY()

CamPitch / CamYaw

Gets current pitch or yaw of the camera.

Parameters: none

Returns: float

local pitch = CamPitch()

GetDeltaTime

Returns time passed since last frame.

Parameters: none

Returns: float

local dt = GetDeltaTime()

SetPhysics

Enables or disables physics for the entity.

Parameters: bool state

Returns: void

entity:SetPhysics(true)

SetDynamic

Sets the entity to be dynamic or static.

Parameters: bool dynamic

Returns: void

entity:SetDynamic(false)

Remove

Removes the entity from the scene.

Parameters: none

Returns: void

entity:Remove()

Log

Logs message console and engine.

Parameters string message

Returns:void

Log("Hello World!")

Accessing Position and Rotation

There are two ways to access position and rotation data in scripts:

  • Global Access: For the current entity the script is attached to: 
    local x = Pos:getX()
local rotY = Rot:getY()
  • Object-Based Access: For other entities retrieved via GetEntity(): 
    local other = GetEntity("Enemy")
local x = other:pgetX()
local rotY = other:rgetY()

Global functions like Pos:getX() only work on the attached entity. If you're working with another entity object, use the pgetX() / rgetY() methods provided through the Entity Object.

Predefined Scripting Functions

Create()

Called once when the entity is created or the script is first loaded. Use this to initialize variables or set up components.

Update()

Called every frame. Use this for continuous logic such as movement, animations, or checks.

OnCollision(otherEntity)

Triggered when the entity collides with another. The otherEntity parameter provides a reference to what was hit.
