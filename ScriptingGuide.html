Flawed Engine Scripting Guide

Scripting Functions

  • Move(float x, float y, float z)

    Moves the object by the specified amount.

    Example

    Move(10.0, 0.0, 5.0)

  • SetPos(float x, float y, float z)

    Sets the position of the object to the specified coordinates.

    Example

    SetPos(5.0, 3.0, 2.0)

  • Rotate(float x, float y, float z)

    Rotates the object by the specified angle around the specified axis.

    Example

    Rotate(0.0, 90.0, 0.0)

  • Scale(float x, float y, float z)

    Sets the scale of the object by the specified factors along each axis.

    Example

    Scale(2.0, 2.0, 2.0)

  • ApplyForce(float x, float y, float z)

    Applies a force to the object along the specified direction.

    Example

    ApplyForce(0.0, 10.0, 0.0)

  • ApplyRelativeForce(float x, float y, float z)

    Applies a force to the object relative to its orientation.

    Example

    ApplyRelativeForce(0.0, 5.0, -5.0)

  • SetPhysics(bool state)

    Set state of physics on object.

    Example

    SetPhysics(true)

  • SetDynamic(bool state)

    Set the dynamic phyxs state of the object.

    Example

    SetDynamic(true)

  • ChangeColor(float r, float g, float b)

    Changes the color of the object to the specified RGB values.

    Example

    ChangeColor(1.0, 0.5, 0.0)

  • MoveCamera(float x, float y, float z)

    Moves the camera by the specified amount.

    Example

    MoveCamera(0.0, 5.0, -10.0)

  • SetCameraPos(float x, float y, float z)

    Sets the position of the camera to the specified coordinates.

    Example

    SetCameraPos(0.0, 0.0, 20.0)

  • ChangeAnimation(string)

    Changes the animation of the object to the specified animation path.

    Example

    ChangeAnimation("Assets/Anims/myAnimation.dae")

  • AddScript(string)

    Adds a lua script to object from path.
    Calls the create function on script.

    Example

    AddScript("Assets/Scripts/myScript.lua")

  • GetEntity(string) - returns entity object

    Returns the entity object with the specified name.

    Example

    player = GetEntity("Player")
    player:SetPos(10, 2, 1)

  • SpawnObject(string, int) - returns entity object

    Spawns an object of the specified type and returns the entity object.

    Example

    cubeObj = SpawnObject("Cube", 0)
    myEntity = SpawnObject("CustomEntity", 7)

    myEntity:ChangeAnimation("Assets/Anims/myAnimation.dae")

  • IsKeyDown(int) - return bool

    Returns true if the specified key is currently pressed; otherwise, returns false.
    Uses the Ascii table for now.

    Example

    if IsKeyDown(75) then
        print("K was pressed!")
    end

  • Get Position - return float

    Sets of function to retrive the location of an Entity.

    Example

    x = Pos:getX()
    y = Pos:getY()
    z = Pos:getZ()
    SetPos(x, y, z)

    someEntity = GetEntity("someEntity")
    eX = someEntity:pgetX()
    eY = someEntity:pgetY()
    eZ = someEntity:pgetY()
    someEntity:SetPos(eX, eY, eZ)

Predfined functions

  • OnCollision(otherEntityName)

    This function is called when an object collieds with current object.

    Example

    function OnCollision(otherEntityName)
        entity = GetEntity(otherEntityName)
        entity:ChangeColor(0, 1, 0)
    end

Object IDs

Used when spawning objects.

  • 0. Cube
  • 1. Sphere
  • 2. Cone
  • 3. Torus
  • 4. Triangle
  • 5. PointLight
  • 6. SpotLight
  • 7. Custom
